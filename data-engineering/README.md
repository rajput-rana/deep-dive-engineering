## ğŸ’¡ Quick Reference

<div align="center">

### Data Lake vs Data Warehouse

| Aspect | Data Lake | Data Warehouse |
|:---:|:---:|:---:|
| **Data Type** | Raw, unstructured | Structured, processed |
| **Schema** | Schema-on-read | Schema-on-write |
| **Cost** | Lower storage cost | Higher processing cost |
| **Use Case** | Exploration, ML | Business intelligence |

### Processing Frameworks

| Framework | Use Case | Speed |
|:---:|:---:|:---:|
| **Map Reduce** | Batch processing | Slower |
| **Spark** | Batch + streaming | Faster (in-memory) |
