# Technical Concepts

Deep dives into core backend and distributed systems concepts.

## Topics

- **Concurrency** - Threading, async programming, race conditions, synchronization
- **Databases** - SQL/NoSQL, indexing, transactions, replication, sharding
- **Networking** - Protocols, HTTP/HTTPS, TCP/IP, DNS, load balancing
- **Distributed Systems** - Consensus, CAP theorem, distributed transactions, consistency models
- **Caching** - Cache strategies, eviction policies, distributed caching
- **Messaging & Streaming** - Message queues, event streaming, pub/sub patterns
- **Security & Auth** - Authentication, authorization, encryption, OAuth, JWT
- **Scalability & Performance** - Performance optimization, bottlenecks, profiling

Each topic folder contains detailed explanations, examples, and best practices.
