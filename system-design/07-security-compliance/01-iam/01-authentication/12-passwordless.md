# Passwordless Authentication

**What it is:** Authentication without passwords.

**Methods:** WebAuthn, Passkeys, Biometrics

## Passwordless Methods

### WebAuthn / Passkeys
- **How:** Browser/OS manages credentials
- **Pros:** Most secure, phishing-resistant
- **Cons:** Requires compatible device

### Biometric Authentication
- **Methods:** Fingerprint, Face ID, Touch ID
- **Pros:** Convenient, secure
- **Cons:** Device-dependent

### Magic Links
- **How:** Email link for login
- **Pros:** No password needed
- **Cons:** Email security required

## Why Passwordless?

- ✅ **No password to steal** - Eliminates password attacks
- ✅ **Better UX** - Faster, easier login
- ✅ **More secure** - Phishing-resistant
- ✅ **Reduced support** - No password resets

## Best Practices

- ✅ Use WebAuthn/Passkeys when possible
- ✅ Implement fallback methods
- ✅ Support multiple devices
- ✅ Clear user communication

## When to Use

- ✅ Modern applications
- ✅ High-security requirements
- ✅ User convenience priority
- ✅ Mobile-first applications
