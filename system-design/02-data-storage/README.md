# Data Storage

Database concepts, storage patterns, and data management strategies.

## Topics

- Database Sharding & Partitioning
- Replication (Master-Slave, Master-Master)
- Indexing Strategies
- SQL vs NoSQL
- ACID vs BASE
- Consistency Models
- Database Types (Time-Series, Columnar, Document, Key-Value)

## Key Concepts

- **Scalability:** Sharding and partitioning
- **Reliability:** Replication and redundancy
- **Consistency:** ACID, BASE, eventual consistency
- **Performance:** Indexing and optimization
