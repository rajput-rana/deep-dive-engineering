1. Requirements Gathering
2. Back-of-the-Envelope Estimation
3. High-Level Design (HLD)
4. Data Model / API Design
5. Detailed Design / Deep Dive
6. Identify Bottlenecks and Trade-offs
7. Review, Explain, and Iterate
