# Authentication & Security

Authentication, authorization, and security patterns for system design.

## Topics

- JWT (JSON Web Tokens)
- OAuth 2.0
- Session Management
- Single Sign-On (SSO)
- Multi-Factor Authentication
- API Security
- Encryption (at rest, in transit)

## Key Concepts

- **Authentication:** Verifying user identity
- **Authorization:** Controlling access
- **Token Management:** JWT, OAuth, refresh tokens
- **Security:** Encryption, secure communication
