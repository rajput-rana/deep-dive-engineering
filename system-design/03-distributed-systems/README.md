# Distributed Systems

Concepts and patterns for building distributed systems that work across multiple machines.

## Topics

- Service Discovery
- Consensus Algorithms (Raft, Paxos)
- Distributed Transactions
- Event Sourcing & CQRS
- Distributed Locking
- Leader Election
- Fault Tolerance
- Consistency Models

## Key Concepts

- **Consensus:** Getting multiple nodes to agree
- **Consistency:** Ensuring data coherence
- **Partition Tolerance:** Handling network failures
- **Fault Tolerance:** Surviving node failures
